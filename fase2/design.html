Propósito del sitio

En este sitio Web vamos a ver lo relacionado con las Etiquetas semánticas y CSS Grid para la creación de sitios Web, se busca que el lector logre comprender de la mejor manera estos temas para asi lograr la creacion exitosa de un Sitio Web Básico.

Etiquetas semánticas

Con las etiquetas semánticas se describe el significado del contenido y permite que los documentos HTML sean más claros para los desarrolladores y para los navegadores que procesan y renderizan sus informaciones. En la versión 5 del HTML, las principales modificaciones son justo en la estructura semántica, con nuevas tags que representan las secciones, con las divs madres.

Los elementos semánticos

La etiqueta header es utilizada para identificar la cabecera de la página.

La etiqueta nav representa el menú de navegación de la página.

El elemento main especifica el contenido principal de la página, o sea, de mayor relevancia para el usuario.

El elemento section representa una parte o una sección de la página.

El elemento article declara un contenido que es independiente de cualquier otras secciones.

El elemento figure es una marcación de uso específico para insertar figuras o imágenes, con sus atributos.

El atributo alt promueve la accesibilidad del contenido porque contiene la descripción de la imagen img de la página y es esa información que el lector de voz lee para el usuario, en los casos de las personas con discapacidad visual.

El elemento aside es utilizado para crear un contenido de apoyo o adicional al contenido principal.

El elemento footer representa el pie de la página, en la área inferior y última de la página.

Fuente: Tatiani Meneghini (2021). La semántica en HTML5. https://www.aluracursos.com/blog/la-semantica-en-html5

CSS Grid

CSS Grid es un sistema de posicionamiento avanzado de CSS en el que ubicamos los elementos HTML de nuestra página web con base a una cuadrícula. Dentro de esta cuadrícula, los elementos se ubican en celdas que podemos modificar usando una variedad de propiedades.

Fuente: Redacción KeepCoding (2022). Cómo funcionan las áreas en CSS Grid - CSS Grid https://keepcoding.io/blog/como-funcionan-las-areas-en-css-grid/
